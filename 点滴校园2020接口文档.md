# 点滴校园2020接口文档

## 访问排名: /sl-2020/login

### `GET`

```json
{
    "code":200,
    "m":"",
    "data":null
}
```


## 弹幕: /sl-2020/bullet-chat

### `GET`请求弹幕

返回60条随机弹幕

```json
{
    "code":200,
    "m":"",
    "data":[
    {
        "avatarUrl":"...............",
        "msg": ".........",
        "type": ""       
    },
    {
        "avatarUrl":"...............",
        "msg": ".........",
        "type": ""       
    }
    ]
}
```

### `POST`提交弹幕


| 参数名称  | 参数类型 | 备注 |
| --------- | -------- | ---- |
| msg | String   | 弹幕内容 |
